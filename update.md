# Update commands

## Jupyter

```zsh
conda activate notebook_env
conda update jupyterlab
jupyter labextension update
```

## Homebrew

```zsh
brew update
```

## Pipx

```zsh
pipx upgrade-all
```

## Poetry

```zsh
poetry self update
```

## Pip

```zsh
pip install --upgrade pip
```

## Conda

```zsh
conda update conda
```

## Oh My Zsh

```zsh
upgrade_oh_my_zsh
```
